Secure Multimedia Steganography

Hide secret messages or files (images, PDFs, documents, audio, video, etc.) in images, audio, and video using AES-256-GCM encryption and LSB steganography. Your data is protected before being hidden in multimedia files.

Features

🔒

Dual-Layer Encryption

Messages are encrypted with AES-256-GCM for strong confidentiality + integrity.

🛡️

LSB Steganography

Data is hidden using the Least Significant Bit technique across all media types.

📺

Multimedia Support

Supports ALL image formats (JPEG, PNG, WebP, BMP, GIF, TIFF, AVIF, etc.), ALL audio formats (MP3, WAV, AAC, M4A, OGG, FLAC, etc. - auto-converted to WAV), and ALL video formats (MP4, AVI, MOV, MKV, WebM, etc.).

How It Works

Encoding

  1. Upload your file (Image/Audio/Video)
  2. Choose: Text message or File to hide
  3. Set encryption password
  4. System encrypts with AES-256-GCM
  5. Download stego file

Decoding

  1. Upload stego file
  2. Enter the password
  3. System extracts hidden data
  4. Decrypts with AES-256-GCM
  5. Shows original message or file
